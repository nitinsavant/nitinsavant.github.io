Latest Project

I'm the co-creator of Conclave, a real-time collaborative text editor built in JavaScript. Conclave uses a Conflict-Free Replicated Data Type (CRDT) to make sure all users stay in-sync and WebRTC to allow users to communicate directly with one another without a central server. The result is a private and secure way for users to collaborate on documents. Read our case study or watch my talk below to learn more.
